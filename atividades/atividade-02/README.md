Instruções da atividade-02 (que está no classroom)

Implementar uma Pilha
1) Entender o funcionamento de uma Pilha;
2) Implementar o código para definir uma Pilha;
3) Implementar as funcionalidades :
a) Mostrar  topo da Pilha;
b) Empilhar;
c) Desempilhar;
d) Mostrar todos os elementos da Pilha.
e) Entregar o código e um print da execução.

Funcionamento da Pilha:
A pilha é uma forma de organização presente nas funções recursivas, por exemplo, na qual uma pilha de pratos se assemelha ao funcionamento da pilha.
O ultimo adicionado é o primeiro a sair, além de outras funcionalidades como mostrar a pilha e onde é seu topo.
